# arcadia
Cet album photo entièrement en javascript permet de lire le contenu des dossiers photos placés dans le dossier "albums".<br />
Il utilise <a href="https://github.com/MikeKovarik/exifr">exifr</a> pour l'affichage des métadonnées contenues dans chaque photo.<br />
Ainsi que <a href="https://github.com/fancyapps/fancybox">fancybox</a> pour l'affichage des photos au format original.<br />
L'affichage des photos se fait au format blog ou mosaïque.
